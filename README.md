# Чистый доход

Минималистичное Android-приложение для учета реального дохода от смен, подработок, доставок и почасовой работы.

Главная цель приложения: за несколько секунд показать, сколько пользователь заработал чистыми и сколько ему еще должны.

## Для кого

- курьеры;
- таксисты;
- официанты;
- мастера;
- продавцы;
- фрилансеры;
- люди с несколькими работами или подработками.

## Что умеет

- считать начисленный доход;
- учитывать расходы;
- показывать чистый доход;
- учитывать выплаты и авансы;
- считать остаток к получению;
- считать чистый доход в час;
- показывать прогресс к цели дохода;
- хранить все данные локально на устройстве;
- экспортировать данные в CSV;
- делать резервную копию в JSON;
- импортировать данные из JSON.

## Beta 0.2.0

Эта версия сфокусирована на реальном ежедневном сценарии:

- главная показывает только чистыми, начислено, расходы, часы, остаток и чистыми в час;
- нижняя навигация упрощена до 4 вкладок с центральной кнопкой `+`;
- добавлена отдельная форма `Курьер / заказы`;
- перед сохранением смены показывается предпросмотр расчета;
- время можно вводить как `16`, `1600`, `16:00`, `16.00`, `16 00`, `930`;
- дата выбирается через DatePicker, есть быстрые кнопки `Сегодня` и `Вчера`;
- суммы принимают `1604`, `1604.86`, `1604,86`, `1 604,86`;
- экран работ показывает только релевантные поля для выбранного типа расчета;
- аналитика без данных показывает одно аккуратное пустое состояние.

Проверочный пример для курьера:

```text
Работа: Магнит курьер
Дата: 30.04.2026
Начало: 16
Конец: 20
Количество заказов: 10
Начислено сервисом: 1604,86
Дополнительные чаевые: 100
Расходы за смену: 0
```

Ожидаемый расчет:

```text
Часы: 4
Заказов: 10
Начислено: 1 704,86 ₽
Чистыми: 1 704,86 ₽
Чистыми в час: 426,22 ₽/ч
Доход за заказ: 170,49 ₽
```

## Основные экраны

- **Онбординг**: создание первой работы.
- **Главная**: только ключевые показатели без перегруза.
- **Добавить**: смена, доход, расход или выплата.
- **История**: список записей, редактирование, удаление, дублирование.
- **Аналитика**: доход по дням, доход по работам, расходы по категориям.
- **Еще**: выплаты, работы и настройки.
- **Настройки**: карточки главного экрана, формулы, цель, экспорт и импорт.

## Расчеты

- **Начислено** = базовая оплата смен + чаевые + бонусы - штрафы.
- **Расходы** = сумма расходов.
- **Чистый доход** = начислено - расходы.
- **Выплачено** = сумма выплат и авансов.
- **Осталось получить** = начислено - выплачено.
- **Чистыми в час** = чистый доход / отработанные оплачиваемые часы.
- **Средний доход за смену** = чистый доход / количество смен.

Для смен учитывается:

- переход через полночь;
- оплачиваемый или неоплачиваемый перерыв;
- почасовая оплата;
- фикс за смену;
- ручной доход;
- заказной режим;
- чаевые, бонусы и штрафы.

## Как открыть в Android Studio

1. Открой Android Studio.
2. Выбери **Open**.
3. Укажи папку проекта:

```text
D:\Codex\clean-income-android
```

4. Дождись синхронизации Gradle.
5. Нажми **Run** для запуска на эмуляторе или телефоне.

## Как собрать APK

Из PowerShell:

```powershell
cd D:\Codex\clean-income-android
.\gradlew.bat assembleDebug
```

Готовый APK появится здесь:

```text
app\build\outputs\apk\debug\app-debug.apk
```

В этом рабочем каталоге также может быть удобная копия для релиза:

```text
CleanIncome-beta.apk
```

## Как установить на телефон через ADB

Подключи Android-телефон по USB, включи USB-отладку и выполни:

```powershell
D:\Android\Sdk\platform-tools\adb.exe install -r D:\Codex\clean-income-android\CleanIncome-beta.apk
```

Проверить, видит ли ПК телефон:

```powershell
D:\Android\Sdk\platform-tools\adb.exe devices
```

## Как поймать баги

Очистить лог:

```powershell
D:\Android\Sdk\platform-tools\adb.exe logcat -c
```

Записать лог приложения:

```powershell
D:\Android\Sdk\platform-tools\adb.exe logcat -v time CleanIncome:D AndroidRuntime:E ActivityManager:E InputDispatcher:W *:S > D:\Codex\clean-income-android\freeze-log.txt
```

После повторения бага останови команду через `Ctrl+C`. Лог будет в файле:

```text
D:\Codex\clean-income-android\freeze-log.txt
```
